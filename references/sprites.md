# Buddy 精灵数据（终端预览用）

当用户询问某个物种长什么样时，读取此文件并渲染对应的 ASCII 精灵。
用猫(cat) + 星星眼(✦)作为默认展示。帽子替换第一行空白行。

## 眼睛映射

| 中文 | 符号 | 描述 |
|------|------|------|
| 默认 | · | 简约，平静 |
| 星星 | ✦ | 闪亮，兴奋 |
| 晕眩 | × | 调皮或晕眩 |
| 大眼 | ◉ | 警觉 |
| 机械 | @ | 数字化，机械感 |
| 惊讶 | ° | 空洞凝视 |
